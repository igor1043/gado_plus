# Gado Plus

Sistema de controle  que facilita a gestão de propriedade, gastos e pesagem de gado de corte para pequenos produtores rurais.


## Pré-requisitos

- Android Studio 3.5.3
- Room 2.2.0
- MPAndroidChart 3.1.0
